Term 04 — Server Infrastructure

Web Servers

Hardware or software systems that store, process, and deliver web pages to clients over HTTP or HTTPS upon request.

📷 Add your image: images/web-servers.jpg

DEFINITION

A web server is a computer system — hardware, software, or both — that stores web content such as HTML pages, images, and files, and delivers them to clients (web browsers) over the Internet using the HTTP or HTTPS protocol whenever a request is made.

How It Works

Web servers like Apache, Nginx, and Microsoft IIS listen for incoming HTTP requests on port 80 or HTTPS requests on port 443. When a request arrives from a browser, the server locates the requested file or generates dynamic content and sends a response back.

The response includes a status code such as 200 OK (success), 404 Not Found (missing page), or 500 Internal Server Error (server problem), along with the requested content that the browser then renders for the user.

Popular Software

Apache, Nginx, Microsoft IIS, LiteSpeed, and Caddy are widely used web servers.

HTTP Port

Web servers listen on port 80 for standard HTTP requests by default.

HTTPS Port

Secure HTTPS connections use port 443 with TLS/SSL encryption.

Response Format

Returns an HTTP status code plus the requested content such as HTML or images.
