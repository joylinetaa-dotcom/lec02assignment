Term 01 — Networking

World Wide Web

The global system of interlinked hypertext documents and multimedia content accessed via the Internet using web browsers.

📷 Add your image: images/www.jpg

DEFINITION

The World Wide Web (WWW) is a system of interlinked hypertext documents and multimedia content accessed via the Internet using web browsers. It was invented by Tim Berners-Lee in 1989 and uses HTTP to transfer data and HTML to format content.

How It Works

When you type a URL into a browser, it sends an HTTP request to a web server. The server returns HTML files which the browser renders as a visual web page. Hyperlinks connect pages across the globe into one seamless web of information.

The Web operates on top of the Internet infrastructure. It is not the same as the Internet — the Internet is the physical network of computers, while the Web is a service that runs on top of it.

Invented By

Tim Berners-Lee in 1989 at CERN, Switzerland.

Protocol Used

HTTP and HTTPS for transferring data between browsers and servers.

Content Format

HTML for structure, CSS for styling, JavaScript for interactivity.

Purpose

Global information sharing and communication over the Internet.
