Term 12 — Markup Language

HTML

Hypertext Markup Language — the standard markup language used to create and structure content on the World Wide Web.

📷 Add your image: images/html.jpg

DEFINITION

Hypertext Markup Language (HTML) is the standard language used to create web pages. It uses a system of tags enclosed in angle brackets to define the structure and meaning of web content, such as headings, paragraphs, links, images, and tables.

How It Works

An HTML document starts with a DOCTYPE declaration followed by html, head, and body tags. The head section contains metadata like the page title and links to stylesheets. The body section contains all visible content that the browser renders for the user.

HTML works together with CSS for styling and JavaScript for interactivity. The browser parses the HTML to build a Document Object Model (DOM) tree which is then rendered as a visual web page on screen. The current standard is HTML5.

Current Version

HTML5 is the latest standard, introducing new semantic elements and multimedia support.

File Extension

HTML files use the .html or .htm file extension and open in any web browser.

Works With

CSS adds visual styling and JavaScript adds dynamic behaviour to HTML pages.

Core Structure

Uses opening and closing tags, attributes, and nested elements to structure content.
