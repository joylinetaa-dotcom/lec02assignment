Term 03 — Client Software

Web Browsers

Software applications that retrieve, render, and display web pages from the Internet, serving as the primary interface between users and the World Wide Web.

📷 Add your image: images/web-browsers.jpg

DEFINITION

A web browser is a software application used to access, retrieve, and display content on the World Wide Web. It interprets HTML, CSS, and JavaScript to render visual web pages that users can interact with. Popular examples include Google Chrome, Mozilla Firefox, Microsoft Edge, and Safari.

How It Works

A browser sends HTTP or HTTPS requests to web servers using a URL entered by the user. The server responds with HTML, CSS, and JavaScript files. The browser's rendering engine parses these files and paints pixels on your screen to display the final page.

Modern browsers also support features like bookmarks, browser history, extensions, private browsing mode, and built-in developer tools that allow web developers to inspect and debug web pages.

Popular Examples

Google Chrome, Mozilla Firefox, Microsoft Edge, Apple Safari, Opera.

Core Function

Renders HTML, CSS, and JavaScript into visual web pages for users.

Protocol Used

HTTP and HTTPS to communicate with web servers across the Internet.

Key Component

The JavaScript engine executes scripts to make pages dynamic and interactive.
