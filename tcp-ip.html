Term 02 — Protocols

TCP/IP

Transmission Control Protocol/Internet Protocol — the suite of communication protocols that defines how data is transmitted across networks, forming the backbone of the Internet.

📷 Add your image: images/tcp-ip.jpg

DEFINITION

Transmission Control Protocol/Internet Protocol (TCP/IP) is the suite of communication protocols used to interconnect network devices on the Internet. TCP handles reliable delivery of data while IP handles the addressing and routing of packets.

How It Works

TCP/IP is a two-layer system. IP handles addressing and routing packets of data between devices. TCP ensures reliable delivery by checking that all packets arrive correctly and in order, retransmitting any that are lost.

Data is broken into small packets, each labelled with source and destination IP addresses. IP routes each packet across routers and networks independently. TCP at the destination reassembles packets into the original message and acknowledges receipt to the sender.

TCP Role

Ensures reliable, ordered delivery of data between applications.

IP Role

Handles addressing and routing of packets across networks.

Current Version

IPv4 is most widely used; IPv6 is the newer standard with more addresses.

Layer Model

TCP operates at the Transport layer; IP at the Network layer.
